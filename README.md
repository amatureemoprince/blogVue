## 项目目录解释

- [src](./src) 项目源码 包括各种组件 页面 路由 等
- [public](./public) 公共资源
- [index.html](./index.html) 入口文件 首次访问时会加载
- [vite.config.ts](./vite.config.ts) vite配置文件
- [tsconfig.node.json](./tsconfig.node.json) ts配置文件
- [env.d.ts](./env.d.ts) 让项目可以识别各种文件
